Preventing Facebook account hack

Dec 25, 2015
2 minute read

Facebook account hacking is on the rise, being the most most popular social networking site, this is a threat to millions of users across the globe who have a lot of personal and private information stored in their accounts. It is difficult to train general users with sophisticated techniques. However, it is important that the users follow simple guidelines and remain aware of the immense risk we are exposed to.

Here, we will go through two really effective features Facebook has provided that can help you keep your account safe.
The first one is Login Notifications. You can configure your account to get notified for any login attempt being made from a device that you have never used to access (login to) your account.

It is simple and easy to use feature. Just choose the notification method you prefer and you are all set to go. Go to Settings, then to Security tab and then edit the Login Notifications.

The second one is Login Approval. You can configure your account to use a secret code before you use a new browser. This is something like using a pin number while using your payment card (ATM/debit/credit card). So, even if someone cracks down your password, this user can not login to your account until he/she knows your security code.

At first, as in case of Login Notifications, go to Settings, then to Security tab and then edit the Login Approvals.

You will receive a text message containing a six digit security on your phone number. Please remember this will be sent to the number that is linked to your Facebook account. You will always have the option of changing the phone number, if you need to. Once you submit the security code, you will get a confirmation that the code is working fine.

If you try editing the Login Approvals after finally submitting the form, the Login Approvals to require security code would be checked.

Once you have done all of this, you can test to check if the feature does work and is really useful.

Use a new browser from the same device (desktop/laptop/tablet/mobile) or from a different device to login to your Facebook account. As soon as you submit the username password, you will be promoted to a form to submit the security code you had received from Facebook. If you enter the code correctly, you will be redirected to your home page, else not.

On the other end, if you check the Facebook home page on the existing session through which you configured the Login Approvals, you should receive a notification that someone has attempted to access your account.

You are also given an option to review the recent login with the geographical position from where the login was attempted. You can always reject the new login by selecting This wasnâ€™t me.. and thus creating a new password for you Facebook account. However, in this case it is you who is making the login attempt, you can go with the second option This was me...

Finally, you are given the option if you would want to save the new browser for the next login.
